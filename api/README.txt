Вся возможная информация о погоде содержиться в структурах

структур мы имеем несколько
1) структура с данными о геопозиции (без массива)
2) структура с данными о текущей погоде (без массива)
3) структура с данными за минуту (61 ебейшая запись формата 0,0,0,0,0) (самая скучна почти не имеет инфы USELESS)
4) структура с данными за час (2 дня -> 48 записей) (тяжелая и полноценная)
5) структура с данными за неделю (8 дней)


во всех структурах кроме структур с пометкой (без массива)
данные идут следующим образом
data1|data2|data3| и тд
